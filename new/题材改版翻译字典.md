# 题材改版翻译字典（Xiuxian → Fallout / Wasteland）

> 本文件是“唯一权威字典（Single Source of Truth）”，用于将项目中的修仙/中文硬编码文本替换为 **Fallout 风格废土题材**英文文本。  
> 适用范围：`services/randomService.ts`（18 宗门/28 任务类型/大量描述）、`constants/*`、`components/*`、`views/*`、AI Prompt、日志文本等。  
> 参考来源（用于校对拼写与语气，不直接复制原文）：[Fallout Wiki](https://fallout.fandom.com/wiki/Fallout_Wiki)、Fallout 4/76 游戏内任务文本语气、以及本项目既有“映射原则”。  

---

## 0. 使用方式（给开发者）

### 0.1 两条硬规则

- **规则 A：先定“术语与语气”，再翻译句子**  
  先用本文档的“术语表 + 命名模板 + 任务语气模板”统一风格，再批量替换散落文本，避免出现一半 Pip-Boy、一半仙侠的混搭。

- **规则 B：不追求 100% Fallout 正史，追求“Fallout 语感 + 一致性”**  
  除少数必须保持官方拼写的词（如 Stimpak、RadAway、Enclave、Brotherhood of Steel），其余可“类 Fallout”原创，只要：简短、硬朗、军事/废土/技术感、能被 UI 快速扫描。

### 0.2 落地流程（推荐）

1. **先跑全局中文扫描**：锁定所有中文来源（服务层 > 常量 > 组件硬编码）。  
2. **把发现的中文短语/名词按类型归类**：货币、任务、派系、掉落、地点、敌人、材料。  
3. **对照本字典选择/生成替换**（优先用“词库表”，次选用“模板生成”）。  
4. **替换后再次扫描**：保证关键路径 **0 中文泄露**。  
5. **最后做“风格一致性复查”**：同一概念在 UI/日志/奖励必须同词同写法（Caps vs Bottle Caps 不混用）。

## 1. 全局术语映射（必须一致）

### 1.1 核心概念（强制）

| 原概念（修仙） | 项目内旧写法/风险点 | 目标废土概念（建议） | 备注 |
|---|---|---|---|
| 宗门 / 门派 | sect / 宗门 | **Faction / Outpost / Company / Gang** | “Faction”用于大组织；“Outpost”用于本地据点 |
| 灵石 / 灵石数 | spiritStones / 灵石 | **Caps**（统一用 Caps） | UI、日志、奖励统一用 Caps |
| 炼丹 | alchemy | **Chem Cooking / Craft Chems** | 不要用 Alchemy（除非做“旧世界黑话”但不推荐） |
| 丹药 | pill | **Chems / Meds** | Stimpak / RadAway 等属于 Chems/Meds |
| 法宝 / 法器 | artifact / weapon | **Relic / Pre-War Tech / Weapon Mod** | 强调“战前科技”与“改装” |
| 渡劫 | tribulation | **Rad Storm / Containment Breach / Exposure Event** | 用“辐射/隔离/暴露”语感 |
| 突破 | breakthrough | **Mutate / Evolve / Upgrade** | 结合系统表现：升级/突变/改造 |
| 修炼 / 打坐 | meditation | **Train / Rest / Recover** | 简短动词优先 |
| 秘境 | secret realm | **Site / Zone / Facility / Vault** | “Vault/Facility”更 Fallout；不要叫 Realm |

### 1.2 写作语气（Fallout 任务文本风格）

**句子结构模板（推荐）**：
- **Objective 风**（UI/任务卡/日志）：动词开头 + 目标 + 地点/条件  
  - “Clear the hostiles from **[LOCATION]**.”  
  - “Retrieve **[ITEM]** from **[LOCATION]**.”  
  - “Escort **[TARGET]** to **[DESTINATION]**.”  
- **Pip-Boy 备注风**（补充描述）：短句、偏冷静、带技术/军用词  
  - “Radiation levels elevated. Proceed with caution.”  
  - “Signal lost. Expect interference.”  

**禁用写法**（会显得“仙侠/网文”）：  
- “Heavenly Dao / Immortal / Cultivator / Spirit Qi / Mystic / Sect”  
  若必须保留作为“旧世界代号”，也要在 UI 上解释为 **Pre-War Codename**，且频率极低。

## 2. 难度/品质词库（用于任务与奖励）

> `randomService.ts` 使用：`difficulty: Easy | Normal | Hard | Extreme`，`quality: Common | Rare | Legendary | Mythic`

### 2.1 难度（Difficulty）

| 系统值 | UI 显示词（建议） | 备注 |
|---|---|---|
| Easy | **Routine** | 日常清理/跑腿 |
| Normal | **Standard** | 常规任务 |
| Hard | **High-Risk** | 高风险 |
| Extreme | **Suicidal** / **No-Return** | 极限任务（谨慎使用夸张词） |

### 2.2 品质（Quality）

| 系统值 | UI 显示词（建议） | 备注 |
|---|---|---|
| Common | **Common** | 保留 |
| Rare | **Rare** | 保留 |
| Legendary | **Legendary** | 保留（Fallout 常用） |
| Mythic | **Prototype** / **Mythic** | 若想更 Fallout 可用 Prototype（实验/原型机） |

## 4. 任务类型字典（28 种 TaskType）

> 目标：替换 `TASK_NAMES_BY_TYPE` 与 `TASK_DESCRIPTIONS_BY_TYPE` 的中文池。  
> 本节给出：**每个 TaskType 的建议英文任务名池（Name Pool）**与**描述语气模板（Description Pool）**。  
> 规则：短句、动词开头、地点/物品用占位符（可在实现时随机填充）。

### 4.1 通用占位符（写描述时用）

- `[LOCATION]`：Abandoned Vault / Ruined Factory / Metro Tunnel / Relay Tower / Super Duper Mart  
- `[TARGET]`：caravan / courier / technician / settler / scout  
- `[ENEMY]`：raiders / feral ghouls / super mutants / robots / wildlife  
- `[ITEM]`：fusion core / med supplies / holotape / parts / caps stash  

### 4.2 任务类型映射表（TaskType → Name Pool & Description Pool）

## 5. 掉落/材料字典（用于 DROP_ITEMS、ITEM_POOLS）

### 5.1 “掉落物”命名规则

- **优先使用**：Parts / Scrap / Components / Cache / Mod / Core / Kit / Holotape  
- **避免**：Herb/Flower 过多（会偏奇幻）。若需要植物，用 “Mutfruit / Glowing Fungus / Tato” 等更 Fallout 的词。

### 5.2 推荐掉落词库（可覆盖 `DROP_ITEMS`）

- Mutant Parts / Bio Sample / Rare Herbs / Med Supplies / Weapon Mod
- Damaged Relic / Pre-War Artifact / Circuit Diagram / Data Holotape
- Fusion Core / Energy Cell / Alloy Plate / Scrap Metal / Rare Mineral

### 5.3 推荐材料池（可覆盖 `ITEM_POOLS`）

- **herbs（更像农作/医用）**：Mutfruit / Glowing Fungus / Bloodleaf / Healing Herb / Brain Fungus
- **materials**：Scrap Metal / Circuit Board / Steel / Tungsten / Rare Mineral / Star Core / Polymer
- **masterAlchemy**：Pure Water / Rare Isotope / Fusion Cell / Quantum Particle

## 6. 地点/秘境字典（用于 REALM_NAMES_*）

> 说明：当前 `REALM_NAMES_BY_RISK` 虽已英文化，但仍偏“仙侠遗迹”。建议后续替换为更 Fallout 的地点词库。

### 6.1 地点命名模板

- **结构**：`[Adjective] + [Noun]` 或 `[Noun] + [Qualifier]`  
  - “Abandoned Vault” / “Collapsed Metro” / “Ruined Factory” / “Relay Tower”  
- **高风险词**（用于 High/Extreme）：  
  - “Irradiated” / “Contaminated” / “Quarantine” / “Blacksite” / “Ground Zero”

### 6.2 示例地点池（按风险）

- **Low**：Abandoned Shack / Farm Ruins / Dry Riverbed / Scavenger Camp / Small Vault Access
- **Medium**：Metro Tunnel / Raider Outpost / Medical Facility / Relay Tower / Armory Cache
- **High**：Irradiated Plant / Blacksite Lab / Toxic Marsh / Super Mutant Nest / Collapsed Reactor
- **Extreme**：Ground Zero / Quarantine Vault / FEV Lab Core / Dead Zone / Containment Breach Site

## 7. 校对清单（避免“看似英文但不 Fallout”）

- **拼写一致**：Stimpak / RadAway / Rad-X / Mentats / Buffout / Psycho / Jet / Med-X（不要变体）  
- **货币一致**：Caps（不混 Bottle Caps / Credits / Spirit Stones）  
- **组织一致**：Faction / Outpost（不出现 Sect）  
- **语气一致**：短句、冷静、军事/生存导向；少用诗意形容词
